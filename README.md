# Movie Ticket Reservation System

## Description
A web-based movie ticket reservation system that allows users to view movies, select seats, and book tickets online.

## Tech Stack
- Frontend: React / HTML / CSS
- Backend: Node.js, Express
- Database: MongoDB

## Features
- User authentication
- Movie listing
- Seat selection
- Ticket booking

## How to Run
1. Clone the repository
2. Install dependencies
   npm install
3. Start the server
   npm start
